# ParasFlix Frontend

A modern, fast, and elegant Vue 3 frontend for the ParasFlix media server, designed to provide a Netflix-like streaming experience.

## Tech Stack
- **Vue 3** (Composition API, `<script setup>`)
- **TypeScript** (Strict mode)
- **Vite** (Next generation frontend tooling)
- **Pinia** (State management)
- **Vue Router** (Routing)
- **Vidstack** (Production-ready media player)
- **Lucide Icons** (Clean, modern SVG icons)

## Architecture

- **`src/api`**: Centralized Axios configuration and endpoint definitions.
- **`src/components`**: Reusable components split by domain (layout, library, player, ui).
- **`src/composables`**: Reusable composition functions.
- **`src/router`**: Route definitions with lazy loading.
- **`src/stores`**: Pinia stores for library, search, and continue watching state.
- **`src/types`**: TypeScript interfaces matching the FastAPI backend models.
- **`src/utils`**: Helper functions (debounce, formatters).
- **`src/views`**: Page-level components.

## Features
- **Responsive Layout**: Works seamlessly across Desktop, Tablet, and Mobile.
- **Lazy Loading**: Images and routes are lazily loaded to minimize initial bundle size.
- **Continue Watching**: Playback progress is locally persisted and automatically restored.
- **HLS Streaming**: Vidstack player natively supports `.m3u8` master playlists generated by the backend.
- **Search**: Debounced search bar with instant results.

## Development

1. Ensure the FastAPI backend is running on `http://localhost:8080`.
2. Run `npm install` to install dependencies.
3. Run `npm run dev` to start the development server.

## Production Build

Run `npm run build`. The resulting `/dist` folder can be served by any static file server (Nginx, Caddy, etc.).
